# 🍲 TheRecipeHub

TheRecipeHub is a modern and responsive recipe website where users can search for recipes, explore cuisines, watch YouTube cooking tutorials, and save favorite recipes.

---

# 🚀 Features

- 🔍 Search Recipes
- 🍕 Explore Multiple Categories
- 🎥 YouTube Cooking Tutorials
- ❤️ Add to Favorites
- 💾 LocalStorage Support
- 📱 Fully Responsive Design
- 🌎 Multiple Cuisine Options
- ⚡ Fast and Dynamic UI

---

# ▶️ How to Run the Project

## Method 1: Directly Open

1. Download the project
2. Open `index.html`
3. Run in browser

---

## Method 2: Using VS Code Live Server

1. Open project in VS Code
2. Install Live Server Extension
3. Right Click `index.html`
4. Click `Open with Live Server`

# 🔍 Functionalities

## Recipe Search
Users can search recipes instantly using the search bar.

## Category Filter
Filter recipes according to cuisine type.

## YouTube Video Integration
Each recipe contains a tutorial video link.

## Favorites System
Favorite recipes are saved using browser LocalStorage.

## Responsive Design
Works on:
- Desktop
- Tablet
- Mobile Devices
